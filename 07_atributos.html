07 - Atributos en HTML

Los atributos HTML aportan información adicional a los elementos, modificando su comportamiento o proporcionando metadatos.

1. Atributos globales

Estos atributos pueden aplicarse a casi cualquier etiqueta HTML:

  • id: Identificador único del elemento.
  • class: Define una o más clases CSS.
  • title: Texto emergente que aparece al pasar el mouse.
  • lang: Define el idioma del contenido.
  • style: Aplica estilos en línea directamente al elemento.

4. Buenas prácticas

  • Siempre usar alt en imágenes para accesibilidad.
  • Usar label for asociado a input id para formularios accesibles.
  • No abusar del atributo style en línea; es mejor usar CSS externo.
