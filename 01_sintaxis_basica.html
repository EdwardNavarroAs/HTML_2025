01 - Sintaxis básica de HTML

HTML (HyperText Markup Language) es un lenguaje de marcado que se utiliza para estructurar el contenido de las páginas web.

Un documento HTML se compone de etiquetas (o elementos) que indican cómo debe presentarse o comportarse cada parte del contenido

  • Las etiquetas normalmente tienen una apertura y un cierre. Ejemplo: <p> ... </p>.
  • Algunas etiquetas son vacías, es decir, no tienen contenido interno ni cierre. Ejemplo: <img> o <br>.
  • Las etiquetas pueden tener atributos que agregan información adicional. Ejemplo: lang="es", class="boton".
  • Algunos atributos son booleanos, como disabled, checked o autoplay.
