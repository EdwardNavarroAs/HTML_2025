Este es un párrafo de ejemplo que forma parte del contenido visible en la página.

HTML es un lenguaje de marcado que define la estructura del contenido web.

Las etiquetas pueden tener atributos como lang="es" que indican el idioma, o charset="utf-8" para definir la codificación.

Algunas etiquetas son "vacías", como <br> o <img>, y no necesitan etiqueta de cierre.
