Énfasis de texto

HTML permite dar énfasis visual y semántico al contenido:

  • <strong>: Indica texto importante (se ve en negrita). Aporta semántica y mejora el SEO.
  • <em>: Indica énfasis (se ve en cursiva). Aporta semántica útil para lectores de pantalla.
  • <mark>: Resalta texto como si estuviera subrayado con marcador. Se usa para destacar coincidencias o resultados en contexto.
  • <b>: Muestra texto en negrita, pero sin semántica. Se usa solo cuando la presentación visual es necesaria, como nombres de productos.
  • <i>: Muestra texto en cursiva, sin énfasis semántico. Se usa para palabras extranjeras, pensamientos, términos técnicos, etc.
  • <u>: Subraya texto. Puede usarse para nombres propios o errores ortográficos, pero debe evitarse para enlaces visuales para no confundir al usuario.
  • <span>: No tiene significado semántico, se usa como contenedor genérico para aplicar estilos con CSS o lógica con JavaScript.
