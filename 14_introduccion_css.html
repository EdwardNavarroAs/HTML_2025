13 - Introducción a CSS

CSS es un lenguaje que permite aplicar estilos visuales a los elementos HTML.

1. ¿Qué es CSS?

CSS significa Hojas de Estilo en Cascada (Cascading Style Sheets) y sirve para controlar el aspecto visual de una página web: colores, fuentes, tamaños, espacios, posiciones y mucho más.

2. HTML, DOM y CSS

Cuando un navegador carga un documento HTML, crea una representación interna del contenido llamada DOM (Document Object Model).

CSS interactúa con este DOM para modificar la apariencia de los elementos, sin alterar su estructura.

3. Formas de aplicar CSS

  • En línea: A través del atributo style en la etiqueta HTML.
  • Interno: Dentro de una etiqueta <style> en el <head>.
  • Externo: Mediante un archivo CSS enlazado con <link>.

4. Estructura de CSS

CSS está formado por reglas que combinan:

  • Selector: Indica a qué elemento aplicar el estilo.
  • Propiedad: Característica visual a modificar (color, font-size, etc.).
  • Valor: El nuevo valor que se asigna a la propiedad.
                p {
                    color: red;
                    font-size: 16px;
                }

La regla anterior hace que todos los párrafos aparezcan en rojo y con tamaño 16px.

5. Buenas prácticas

  • Prefiere siempre CSS externo para mantener el código HTML limpio y separado del estilo.
  • Evita estilos en línea, ya que dificultan el mantenimiento.
  • Utiliza selectores semánticos y clases descriptivas.
